## What this is

ChatKit is a ready-made chat UI for your agent. Think: “the front desk” for your agent.

## Why it helps

- faster launch
- consistent UX patterns
- fewer UI edge cases

<Callout kind="tip" title="Make it feel like your product">
Even a great agent feels suspicious if it looks like a random widget. Branding and copy matter for trust.
</Callout>

<details>
  <summary>Builder notes (optional)</summary>

Treat the UI like an instrument panel. Show tool actions, show citations, and make approvals obvious.
</details>
